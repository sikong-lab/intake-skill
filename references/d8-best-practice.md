# D8 最佳实践（关键维度详解）

D8直接决定方案方向：有最佳实践→补断点；没有→从零设计。**搞反了会导致严重返工。**

## 四种状态

### 1. 有且知道
客户明确说"我之前这样做过，效果很好"
→ **保护它，在上面补。** 不要推翻客户已经跑通的东西。

### 2. 以为有但没有
客户觉得自己有套路，但细问发现跑不通。
→ **追问细节让对方自己发现漏洞。** 不要直接说"你那个不行"。

### 3. 不知道自己有
客户手上的散装零件拼一拼其实就是完整方案。
→ **帮他看到"你其实已经有了"。** 这种情况客户会很惊喜。

### 4. 确实没有
真的从零开始。
→ **标记"无参照"**，方案方向=从零设计。

## 追问D8的话术

- "你之前有没有做过类似的？哪一次你觉得做得最好？"
- "你手上这些文件里，有没有哪个是你觉得'这个就是对的'的？"
- "你说做过第1课了——那第1课你觉得满意吗？格式对吗？内容够不够？"
- 如果对方说"都不太行"，继续追："那你见过别人做的好的吗？有没有参考的样本？"

## 为什么D8最重要

张容容案例教训：她手上有完整的第1课模板（有且知道），但我们没问出来，按"从零设计"的方向做了三次，全部翻车。如果一开始就问"你有没有已经做好的样本"，一次就对了。
